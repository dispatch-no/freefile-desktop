## BUILD ##

### Prerequisites ###

- Qt5
- cmake
- [libsearpc](https://github.com/dispatch-no/freefile-libsearpc)
- [freefile](https://github.com/dispatch-no/freefile-daemon)

### INSTALL ###

```
cmake .
make
make install
```

> Qt 5.2 or higher is required

### INSTALL with Qt5 ###

```
cmake .
make
make install
```

> Qt 5.5 or higher is recommanded but not required
